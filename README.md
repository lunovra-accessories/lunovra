## 🛠 المواصفات التقنية (Tech Stack)

بصفتي المطور، اعتمدت في هذا المشروع على تقنيات تضمن السرعة الاستثنائية:
* **Frontend:** Vanilla JavaScript (ES6+), HTML5, CSS3.
* **Icons:** FontAwesome 6.0 Integration.
* **Data Handling:** Browser LocalStorage (لحفظ السلة بدون قاعدة بيانات).
* **Deployment:** GitHub Pages (استضافة مستقرة ومجانية للأبد).

## 📋 دليل الإدارة السريع (Owner's Dashboard)

لتسهيل إدارة المتجر يدوياً، قمت بتنظيم الكود كالتالي:
1. **تغيير رقم الاستقبال:** ابحث عن `const phone` في السكريبت.
2. **تحديث المخزون:** مصفوفة `defaultProducts` تحتوي على (الاسم، السعر، الرابط).
3. **تغيير الصور:** استخدمت روابط **Unsplash** لضمان جودة عالية وحجم ملف صغير.
